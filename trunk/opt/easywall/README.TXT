In this folder you will find some scripts that are not finished yet!
They are work in progress and are not documented at all for now.

gentoo.init.sh 	- init.d script for Gentoo based distros.
easywalld 	- a perl server wich later will be used to send updates to the firewall from a web interface, witch is still not finished.
getfailedssh.pl - is a perl sshd.log analyzer wich generates a iptables.sh file for blocking connections from IPs wich have failed ssh login attempts to your server(EXPERIMENTAL DO NOT USE!!!).
